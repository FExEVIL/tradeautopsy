## Chrome Web Store Requirements

- **Small Tile**: 440×280px (PNG or JPEG)
- **Large Tile**: 920×680px (PNG or JPEG) 
- **Screenshots**: 1280×800px or 640×400px (PNG or JPEG, max 5)
- **Icon**: 128×128px (included in extension package)

## Upload Order

1. Go to [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole)
2. Upload `dist/tradeautopsy-extension-v1.0.0.zip`
3. Add promotional images:
   - Small tile → `small-tile.png`
   - Large tile → `large-tile.png`
   - Screenshots → `screenshot-1-green.png`, `screenshot-2-violation.png`, `screenshot-4-warning.png`
4. Complete listing details
5. Submit for review
