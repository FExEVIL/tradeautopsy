# Extension Icons

Place your extension icons in this directory:

- `icon-16.png` - 16x16 pixels (toolbar icon)
- `icon-48.png` - 48x48 pixels (extension management page)
- `icon-128.png` - 128x128 pixels (Chrome Web Store)

## Icon Design Guidelines

- Use TradeAutopsy branding colors (emerald green #10b981)
- Keep it simple and recognizable at small sizes
- Use a transparent background
- Consider using the TradeAutopsy logo or a chart/graph icon

## Quick Icon Generation

You can create icons using:
1. **Online tools**: 
   - [Favicon.io](https://favicon.io/)
   - [RealFaviconGenerator](https://realfavicongenerator.net/)
   - [IconKitchen](https://icon.kitchen/)

2. **Design tools**:
   - Figma
   - Adobe Illustrator
   - Canva

3. **AI tools**:
   - DALL-E
   - Midjourney
   - Stable Diffusion

## Temporary Placeholder

For testing, you can use any PNG images with the correct dimensions. The extension will work without proper icons, but they should be replaced before publishing.

## Icon Specifications

- **Format**: PNG with transparency
- **Dimensions**: 
  - 16x16px (icon-16.png)
  - 48x48px (icon-48.png)
  - 128x128px (icon-128.png)
- **Background**: Transparent
- **Colors**: Match TradeAutopsy theme (emerald green, dark background)
